**E是指数的意思**

如：

```
7.823E5 = 782300
```
这里E5表示: 10的5次方。

E代表的英文是exponent，有时也可用index number来表示。

如：
```
1.3565656565E04 = 13565.656565
```
E04是: 乘以10的4次方, 就是13565.656565

如：
```
-7.41025E3 = -7.41025 * 10^3
```
如：
```
1.401298E-5 = 1.401298 *10^(-5)
```
